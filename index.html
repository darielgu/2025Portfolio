Cash Compass

AI-powered financial expense-tracking application.

Cash Compass is an innovative financial tool developed to help users track their spending habits and gain actionable insights for better financial management. The project features user authentication for secure advisor logins, CSV file input for expense analysis, and a robust database solution to securely store and retrieve financial transactions. By leveraging Python and Jupyter Notebook for exploratory data analysis, Cash Compass offers trend identification and decision-making support through intuitive data visualizations and client-specific insights.

  • Client: LPL Financial Hackathon
  • Category: Financial Technology

Belto World Chatbot

AI-powered chatbot for seamless communication.

The Belto World Chatbot is a full-stack AI-powered application designed to deliver a seamless and intelligent chat experience. The backend, built with Express.js and Node.js, integrates securely with a React front-end to provide real-time AI-driven responses. Using the CORS framework, the chatbot ensures secure communication between the front and back ends. A local MongoDB database was developed to store chat history, using a custom JSON data structure for managing user interactions and maintaining a continuous conversation flow. The chatbot leverages the GPT-3.5-turbo model for generating responses, with support for both streaming and non-streaming responses to optimize user satisfaction.

  • Client: Open Source
  • Category: AI Chatbot Development

Fraternity Newsletter

A web application for managing fraternity communications.

The Fraternity Newsletter is a fully functional web application designed to send weekly updates to over 140 fraternity members. The front end was built using React, while the backend uses server-side APIs integrated with a MongoDB database to store and manage email addresses. Additionally, the application connects with the MailChimp API to facilitate automated email distribution, ensuring timely and efficient communication. This project significantly improved the fraternity's internal communication and streamlined the process of sharing updates and announcements.

  • Client: Fraternity Members
  • Category: Web Development
